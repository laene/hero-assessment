# hero-assessment

## Requirements:
Using the publicly available data at https://jsonplaceholder.typicode.com/, build a webpage that displays a React app which does the following:

Displays all users' posts
Provides the ability to filter the posts by user
- Display a dropdown menu that allows a user to filter by checking/unchecking users names in the list. Provide a Select All/Unselect All option
- Consider accessibility in your design and build of the UI
You can use create-react-app to get started.
You can provide styling any way you see fit, but please use SASS or LESS
The presentation is up to you, but please build/design mobile-first.
